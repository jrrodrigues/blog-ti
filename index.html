MCV - Modelo-Visão-Controle

MVC é a sigla para Model-View-Controller, que em português significa Arquitetura Modelo-Visão-Controle. É um padrão de arquitetura de software que divide uma aplicação em três componentes principais: Modelo, Visão e Controlador

Arquivo batch do DOS/Windows

Um arquivo batch é um arquivo texto contendo linhas com comandos que podem ser executados sequencialmente pelo interpretador de comandos do MS-DOS, Windows ou OS/2. São identificados pelas extensões .bat ou .cmd.

GIT - sistema de controle de versões

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.
